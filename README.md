# Shresth's Personal Portfolio

A sleek, modern, and highly interactive developer portfolio utilizing Next.js, Aceternity UI, and Tailwind CSS. Built to showcase projects, skills, leadership roles, and technical engineering acumen.

## 🚀 Live Demo

*You can deploy this to Vercel/Render to provide your live link here.*

## ⚡ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router format)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **UI Components**: [Aceternity UI](https://ui.aceternity.com/) using Three.js for 3D elements.
- **Typography**: Space Grotesk via `next/font/google`
- **Hosting**: Designed for zero-downtime deployment on platforms like Vercel.

## 🛠️ Features

- **Dynamic Aurora Background**: A premium, GPU-accelerated neon cyan & emerald fluid background mesh providing a deeply immersive thematic experience.
- **Bento Grid Layout**: Organizes achievements, technical skills, and experience into beautifully stacked component grids that perfectly scale on mobile and desktop.
- **Custom Interactive Cursor**: A pure physics-driven global cursor trailer that blends over page elements flawlessly for a high-end feel.
- **Modern Project Showcases**: Pinned 3D card components highlighting external GitHub repositories and Live Sites seamlessly.

## 💻 Getting Started Locally

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm installed on your machine.
- Node.js (v18+)
- npm (v9+)

### Installation

1. Clone the repo
   ```sh
   git clone <your-github-repo-link>
   ```
2. Navigate into the directory
   ```sh
   cd portfolio-main
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```

You can now view the application at `http://localhost:3000`.

## 📈 Deployment

You can seamlessly deploy this Next.js project by linking your GitHub repository to platforms like **Vercel** or **Render**.
